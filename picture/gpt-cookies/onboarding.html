Welcome to ChatGPT Cookies

A few things to note before using the extension:

  1. ChatGPT Cookies only works on the openai.com domain (including sub-domains e.g. chat.openai.com).
  2. ChatGPT Cookies does not track, store, or transmit any user data whatsoever. 🔒️ 100% private by default.
  3. ChatGPT Cookies requires certain permissions to work properly, including access to the active tab, cookies, and clipboard (write-only).
  4. ChatGPT Cookies will show a default popup when the user clicks the extension's icon.
  5. ChatGPT Cookies is open-source, which means anyone can freely view or contribute to the source code. For bugs or feature requests, please submit an issue on GitHub or send me a message on Twitter. 🙂
